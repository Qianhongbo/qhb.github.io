| WORDS                 | MEANING                |
| --------------------- | ---------------------- |
| omnipresent           | 无处不在的，普遍存在的 |
| not much              | 不怎么样               |
| The food wasn't much. | 食物不怎么样           |
| not much to look at   | 看上去不怎么样         |
| descriptive           | 描写的                 |
| evocative             | 引起共鸣的             |
| bland                 | 清淡的，枯燥的         |
| innocuous             | 无害的                 |
| innocuous-seeming     | 看似无害的             |
| potent                | powerful，最强的       |
| top                   | v 处在最高位置         |
| league                | 联赛                   |
| league table          | 排行榜                 |
| account for           | 解释、占...比例        |
| do things (to sb.)    | 产生某种强烈的影响     |
| subtle                | 微妙的，无法感知的     |
| miraculous            | 神奇的，不可思议的     |
| immediately           | 明显地                 |
| signal                | 表示                   |
| clinch                | 达成...                |
| clincher              | 起决定性的事情         |
| context               | 语境                   |
| utterance             | 说话，发生             |
| hook                  | 鱼钩，噱头             |
| quizzical             | 疑问的，好奇的         |
